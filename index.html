Zephir

  • Type system: dynamic/static
  • Memory safety: no direct memory management
  • Compilation model: AOT
  • Memory model: task-local garbage collection

AOT vs. JIT

Possible false dilemma?

Ahead of Time

Just In Time

Use Cases

  • Obfuscation
  • Performance
  • Portability

Where Zephir will not help

  • Code that rely on Blocking IO
  • Code that has Heavy Churn
  • Code that makes calls to PHP source

Syntax

  • Structurally Similar to PHP
  • But variables must be declared(types not required)
  • All code must live in Classes and be contained by a Namespace

Example

    let name = "Paul";
    var middleInitial = "K"

Getting Started

  • Install Zephir and Zephir Parser
  • Use the zephir cli to `zephir init <namespace>`
  • Build with `zephir build`
